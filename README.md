# opencv_face_COWA
Python基于opencv人脸识别的考勤系统——by：ZYS
使用说明：
1.打开摄像头
2.进行人脸录入（第一次必须输入id和选取一张识别照片）
3。进行建模学习
4.进行考勤
5.进行之后的录入后不用再选取当前人员的识别照片
